# Correct Lambda Function Update Command

The Lambda function name is: k8s-evaluation-function
(NOT k8s-evaluator)

## From PowerShell (AWS Learner Lab):

cd path\to\k8s-assessment-framework

aws lambda update-function-code `
  --function-name k8s-evaluation-function `
  --zip-file fileb://evaluation/lambda/lambda-deployment.zip
